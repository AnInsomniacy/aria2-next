# Code of Conduct

Aria2 Next follows the Contributor Covenant Code of Conduct, version 2.1, adapted for this repository.

## Expected Behavior

Use respectful, professional language. Focus criticism on code, design, evidence, and project impact. Assume good faith while requiring reproducible facts for technical claims.

## Unacceptable Behavior

Harassment, personal attacks, discriminatory language, doxxing, threats, sexualized attention, spam, and sustained off-topic disruption are not acceptable in project spaces.

## Scope

This policy applies to GitHub issues, pull requests, release comments, and any other project-controlled communication channel.

## Enforcement

Maintainers may edit or remove comments, close issues or pull requests, restrict participation, or block users when behavior harms the project or community.

Report conduct concerns through GitHub issues only when the report can be public. For private reports, contact the maintainer through the security reporting path in `docs/SECURITY.md` and mark the report as conduct-related.
